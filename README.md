# MiniMUD
Play at [minimud.github.io](http://minimud.github.io)

## Building

### Prerequisites

-   Node & NPM

### Install

-   Clone repo
-   `npm install`

### Shorcuts

-   `npm run start` start development server
-   `npm run build` build to ./build
-   `npm run lint` run linter
-   `npm run test` run test suites
